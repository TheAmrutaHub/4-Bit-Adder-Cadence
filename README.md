# 4-Bit-Adder-Cadence
This project implements a 4-bit Binary Adder using Verilog HDL. The Full Adder is designed using dataflow modeling, and four Full Adder modules are connected to create the 4-bit adder. The design is simulated in the Cadence environment with waveform verification.

⚙️ Tools & Technologies

Verilog HDL
Cadence Simulation Environment
Digital Logic Design Concepts
