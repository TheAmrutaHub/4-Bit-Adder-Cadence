This project implements a 4-bit Binary Adder using Verilog HDL.
The design was created and simulated in the Cadence simulation environment, demonstrating the basic arithmetic operation of binary addition used in digital systems.

The adder is built using structural modeling, where four Full Adder modules are connected together to perform 4-bit binary addition.

⚙️ Tools & Technologies
Verilog HDL
Cadence Simulation Environment
Digital Logic Design Concepts

📂 Project Structure
4_bit_adder.v/ → Verilog source code

4_bit_tb.v/ → Testbench code

screenshots/ → Simulation outputs & waveform
🔑 Features

4-bit binary addition
Carry input (Cin) and carry output (Cout)

Structural design using Full Adders
Verilog testbench for functional verification
Simulation waveform analysis

How to Run

Clone this repository

git clone

Compile and run the testbench (adder_tb.v)
